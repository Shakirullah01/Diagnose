Что это за система

Этот сервис создан для родителей, чтобы понятнее увидеть сильные стороны и те навыки, которым нужно уделить больше внимания. Вопросы подбираются с учетом возраста ребёнка.

Для кого
Для родителей детей от 0 до 42 месяцев.
Какая польза
Понятный разбор и простые рекомендации.
Без стресса
Нет «правильно/неправильно». Есть наблюдения.

  • • Ответьте на вопросы

Как это работает

Просто следуйте шагам по порядку
1. Зарегистрируйтесь или войдите
Чтобы сохранять ваши ответы.
2. Добавьте профиль ребёнка
Укажите возраст и данные.
3. Выберите подходящий опрос
С учетом возраста.
4. Ответьте на вопросы
Один вариант на каждый вопрос.
5. Получите результат
Понятный разбор по сферам.
6. При необходимости отправьте специалисту
Только если вы этого хотите.
